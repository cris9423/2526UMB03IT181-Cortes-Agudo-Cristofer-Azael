

# Actividad: Problema de las 8 reinas (Backtracking)

**Autor:** CRISTOFER AZAEL CORTES AGUDO

**Fecha:** 21/04/2026

**Parcial:** 2

**Unidad:** 3

---

## Descripción

Implementación del clásico problema de las 8 reinas usando backtracking. El programa busca una colocación de 8 reinas en un tablero de ajedrez 8x8 de modo que ninguna se ataque entre sí.

## Objetivo

Practicar técnicas de búsqueda y backtracking (recursión y retroceso) para resolver problemas de colocación con restricciones.

---

## Archivos

*   **OchoReinas.java:** Contiene la lógica de backtracking para colocar reinas, comprobaciones de seguridad y método de impresión del tablero.
*   **Main.java:** Inicializa el tablero y ejecuta la solución, mostrando la configuración encontrada o un mensaje si no existe solución.
